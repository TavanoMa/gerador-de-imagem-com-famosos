# Gerador de Imagem com famosos

Ferramenta de geração de imagens com pessoas famosas, contém login e registro de usuários, desenvolvido em Next e integrado ao banco de dados do Supabase.

## Funcionalidades
- Autenticação de usuário
- Seleção do famoso na Home
- Envio do prompt para solicitação de imagens
- Limite de 2 imagens por usuário

## Armazenamento Usuários
Todos os dados dos usuários são salvos em um banco de dados no Supabase, garantindo que os dados não sejam perdidos ao recarregar ou fechar a página.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript + Next
- Supabase
